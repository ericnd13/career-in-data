# GIT BRANCH nome-da-branch
cria uma nova linha do tempo.

# GIT CHECKOUT nome-da-branch
entra na branch selecionada.
- GIT SWITCH nome-da-branch ( mais moderno )

# GIT SWITCH -C nome-da-branch
cria e entra na nova branch

# GIT MERGE nome-da-branch
antes ir para main, para depois dar o merge e juntar a branch com a main.
