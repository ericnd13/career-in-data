# GIT INIT
cria o repositório na pasta/caminho atual. (usar comando uma única vez, setar caminho com cd)

# GIT STATUS
mostra as mudanças que ocorreram desde o último commit.

# GIT ADD
usado para adicionar arquivos para enviar ao github/commitar, área de preparação. (Staging Area)
- git add . seleciona tudo.
não salva nada ainda. apenas marca o que será salvo.

# GIT COMMIT
salva uma nova versão do projeto no histórico local do Git.
- salva localmente.
como se fosse uma fotografia tirada do projeto.

# GIT PUSH
envia os arquivos locais para salva-los na nuvem.
envia os commits locais para o GitHub.
- salva na nuvem.

# GIT REMOTE ADD ORIGIN
conecta projeto local a um repositório no github.
- quando enviar o push, será para esse endereço.
